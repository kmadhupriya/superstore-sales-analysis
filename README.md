# # Superstore Sales Analysis

## Project Description
This project analyzes 4 years (2015-2018) of retail sales data from a Superstore dataset 
to uncover revenue trends, top-performing product categories, and regional sales patterns. 
The goal was to practice real-world data analysis skills — cleaning, aggregating, and 
visualizing data to derive actionable business insights, similar to tasks a Data Analyst 
would perform.

## Problem Statement
Identify which categories, regions, and time periods drive the most revenue, and provide 
data-backed recommendations to improve business performance.

## Data Source
Superstore Sales Dataset — [Download from Kaggle](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final)
